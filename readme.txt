Response Image for Elementor

== Description ==
Responsive Images for Elementor. There is no responsive image tag within Elementor. Elementor does allow for responsive background images, but has no native responsive img or picture tags. Finally, take advantage of responsive img and picture tags with support for lazy loading, dynamic tags (Elementor Pro Required), orientation media queries and more!

1. Ability to add mobile, tablet, and desktop images.
2. Ability to select the tag to use: `img` or `picture`.
3. Ability to add the orientation media query when using the picture tag.
4. Ability to set the default loading behavior for the image: eager or lazy.
5. Ability to use Elementor Pro dynamic tags.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
